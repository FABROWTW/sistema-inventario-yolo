# sistema-inventario-yolo
Sistema de gestión de inventarios basado en visión artificial (YOLO) para una PYME de carpas



## 📝 Resumen de la Reunión con el Cliente
**Cliente:** Dueño de comercializadora de lonas (PYME).
**Objetivo:** Digitalizar el control de stock de lonas (carpas), mallas sombra y estructuras completas usando visión artificial.

### Problemas detectados:
* Confusión en el conteo manual de rollos de lona por color.
* Error al descontar "combos" (cuando se vende la estructura con su lona).
* Falta de visibilidad del stock en tiempo real desde el celular.

## 📋 Requerimientos del Sistema
### Funcionales:
1. **Detección con YOLO:** Identificar lonas azules, naranjas y negras automáticamente.
2. **Gestión de Combos:** Descontar estructura + lona en una sola venta.
3. **Alertas:** Notificar cuando queden pocos rollos de malla sombra.

### No Funcionales:
* La app debe ser rápida en celulares de gama media.
* Debe funcionar en la bodega (modo offline).
